# ⚡ Maha Energy Portal

A modern and interactive renewable energy web platform built to present energy solutions, services, and projects through a clean, responsive, and user-friendly interface.

## 🌐 Overview

Maha Energy Portal focuses on delivering a smooth digital experience with modern UI design, interactive components, and scalable frontend architecture.

## ✨ Features

✅ Modern responsive design  
✅ Interactive user interface  
✅ Smooth animations & transitions  
✅ Project showcase  
✅ Services section  
✅ Skills & technology showcase  
✅ Mobile-friendly layout  
✅ Clean component-based architecture  

## 🛠️ Tech Stack

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/mayuri123-bit/Maha-Energy-portal.git
```

### Navigate to Project

```bash
cd Maha-Energy-portal
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Now open:

```
> “Run locally using npm run dev(localhost)”
```
